Manage Sites

Add AI sites that you want available in the Side Panel selector. Use HTTPS URLs (e.g., https://chatgpt.com/). Some sites may not load in an iframe and will require “Open in Tab”.

Label URL Actions

Embedding caveat: Many providers set X-Frame-Options or frame-ancestors in CSP to prevent embedding. The extension includes limited header rules for a few default hosts, but custom hosts may still refuse loading. Use “Open in Tab” when needed.
